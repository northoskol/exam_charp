# Описание программы на C#: Фильтрация массива строк

## Описание
Эта программа на C# предназначена для фильтрации массива строк, выбирая из него только те строки,
длина которых меньше или равна 3 символам.
Программа позволяет пользователю ввести исходный массив строк с клавиатуры и выводит как исходный массив,
так и новый массив строк, удовлетворяющий условию.

## Инструкции по использованию

1. Запустите программу.
2. Введите элементы массива строк, разделяя их пробелами, и нажмите Enter.
3. Программа выполнит фильтрацию и выведет исходный и новый массив строк.

## Пример

Введите элементы массива (через пробел): This is a test of the program
Вывод:

Исходный массив: [This, is, a, test, of, the, program] -> Новый массив: [is, a, of, the]

## Требования к окружению

Для запуска программы необходимо иметь установленную среду выполнения C# (например, .NET Framework или .NET Core), 
которая поддерживает компиляцию и выполнение C# кода.
